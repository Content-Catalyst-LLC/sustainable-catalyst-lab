# Changelog

## 0.2.0

- Added unified scientific observation board across six official source connectors.
- Added reusable Dataset Inspector with CSV/JSON import, filtering, tables, summaries, SVG plots, project storage, and CSV export.
- Added Source Registry with coverage, temporal scope, format, licensing notes, and connector health.
- Expanded project schema with datasets, observations, saved queries, source snapshots, citations, and map views.
- Expanded NASA GIBS climate maps with opacity, coordinate capture, metadata export, snow, and chlorophyll layers.
- Added telescope filters and observation summaries.
- Added marine taxon summaries and depth visualization.
- Added observation, dataset, evidence, notebook, experiment, and application-routing actions.
- Preserved v0.1.x browser-storage compatibility.
