# Changelog

## 0.4.0 — Physics, Electromagnetism, and Particle Physics Laboratory

- Added a first-class Physics Laboratory module with ten discipline workspaces.
- Added 35 physics methods spanning mechanics, waves, thermodynamics, fluids, optics, electromagnetism, circuits, quantum, nuclear, particle, detector, and measurement analysis.
- Added electric-field resolution for multiple point charges, capacitor networks, magnetic-field models, Lorentz motion, Faraday induction, electromagnetic propagation, skin depth, and rectangular-waveguide cutoff.
- Added RLC impedance and AC power, resonance sweeps, and first-order RC filter analysis.
- Added a structured particle reference for leptons, quarks, bosons, baryons, and mesons.
- Added relativistic kinematics, invariant mass, two-body decay, time-of-flight, track momentum, event significance, and cross-section methods.
- Added nuclear decay, activity, mass defect, and binding-energy methods.
- Added physics experiment templates, notebook actions, project saving, and documentation integration.
- Added physics-specific project collections while preserving all existing project storage keys.
- Added focused `[sc_lab_physics]` shortcode.

## 0.3.0 — Chemistry Laboratory and Spectrometry Expansion

- Expanded Chemistry into composition, reactions, solutions, acid–base, thermochemistry, electrochemistry, kinetics, and calibration workspaces.
- Added advanced Spectrometry Studio processing, peak characterization, calibration, reproducible histories, and project records.

## 0.2.0 — Live Scientific Data and Observation Workspace

- Added the combined observation board, Dataset Inspector, expanded climate maps, telescope filters, marine summaries, source registry, and cross-application record routing.

## 0.1.0 — Scientific Workspace and Project Foundation
