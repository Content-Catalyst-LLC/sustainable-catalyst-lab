# Changelog

## 0.3.0 — Chemistry Laboratory and Spectrometry Expansion

- Reorganized Chemistry into periodic table, composition, reactions, solutions, acid–base, thermochemistry, electrochemistry, kinetics, and calibration workspaces.
- Added percent composition, empirical formula, and molecular formula analysis.
- Added structured reaction records with balanced equations, conditions, and observations.
- Added molarity, molality, mass-percent, grams-per-liter, dilution, and Ksp solubility calculations.
- Added strong acid/base, weak acid/base, buffer, and titration calculations.
- Added calorimetry, Gibbs free energy, equilibrium-constant, Hess-law, Nernst, electrolysis, Arrhenius, and integrated-rate-law calculations.
- Added analytical calibration with linear regression, R², residual standard deviation, LOD, LOQ, and unknown estimation.
- Expanded Spectrometry Studio with method profiles, raw-data retention, processing history, two baseline methods, two smoothing methods, three normalization methods, derivatives, absorbance/transmittance conversion, peak prominence, FWHM, centroid, noise estimates, calibration, project saving, notebook handoff, and CSV export.
- Added project collections for chemical records, reactions, spectra, calibrations, and methods.
- Preserved all v0.1.x and v0.2.0 browser projects through in-place schema normalization.

## 0.2.0 — Live Scientific Data and Observation Workspace

- Added combined scientific observation board.
- Added Dataset Inspector, source registry, telescope filters, marine summaries, and expanded climate-map records.
- Added project datasets, observations, saved queries, source snapshots, citations, and map views.

## 0.1.1 — Scientific Interface and Workspace Integration Patch

- Added command search, workflow navigation, live overview, activity ledger, interactive traceability, and mobile navigation.

## 0.1.0 — Scientific Workspace and Project Foundation

- Added the initial Lab application, public science connectors, climate imagery, periodic table, chemistry foundations, spectrometry foundation, calculators, experiments, notebook, evidence, documentation, and cross-application routing.
