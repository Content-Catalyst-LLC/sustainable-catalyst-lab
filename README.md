# Sustainable Catalyst Lab v0.2.0

An interactive WordPress scientific workspace for live observations, scientific computation, project evidence, experiments, notebooks, and data-connected documentation.

## Primary modules

- Multi-source scientific Observation Board
- Climate and Earth-observation map workspace
- Space telescope and marine biodiversity workspaces
- Scientific Dataset Inspector
- Scientific Source Registry and connector health
- Chemistry, periodic table, stoichiometry, spectrometry, and 34 calculators
- Experiments, evidence, notebook, traceability, and documentation

Install the plugin ZIP and mount the application with `[sc_lab_app]`.

The browser project model preserves v0.1.x storage keys and migrates records to schema version 0.2.0 when loaded or saved.
