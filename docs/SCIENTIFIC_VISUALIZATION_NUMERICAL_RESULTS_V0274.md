# Scientific Visualization for Numerical Results v0.27.4

Sustainable Catalyst Lab v0.27.4 adds a governed visualization layer for numerical results. The Python Compute Core normalizes and downsamples time series, spectra, uncertainty histograms, sensitivities, parameter sweeps, convergence histories, interpolation results, and bounded two-dimensional grids.

The browser renders accessible SVG with a title, description, keyboard focus, and a tabular fallback. SVG, PNG, CSV, and JSON exports are generated without executing arbitrary plotting code.

## Profiles

- Time series
- Frequency spectrum
- Uncertainty distribution
- Sensitivity tornado
- Parameter sweep
- Convergence diagnostics
- Scatter and fitted line
- Heatmap and contour projection

The WordPress workspace is available under **Analyze → Scientific Visualization** and through `[sc_lab_numerical_visualization]`.
