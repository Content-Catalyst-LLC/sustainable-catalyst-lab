# Sustainable Catalyst Lab v0.30.0 — Reproducible Experiment Framework

The experiment framework turns project experiments into governed protocols, runs, replication records, comparisons, and reports. Protocols freeze hypotheses, variables, controls, procedures, sample plans, linked sources, and prespecified analysis before execution. Run records retain results, deviations, environment context, checksums, and replication identity.

## Storage boundary

Experiment records remain in the browser-local project architecture. The Python Compute Core normalizes, validates, hashes, compares, and formats records but is not a permanent server-side experiment database.

## Safety

The framework supports documentation and reproducibility. It does not replace domain-specific laboratory safety, ethics, regulatory, clinical, animal, environmental, or human-subject review.
