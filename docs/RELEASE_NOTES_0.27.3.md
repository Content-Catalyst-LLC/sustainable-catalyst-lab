# Sustainable Catalyst Lab v0.27.3

## Numerical Precision and Solver Governance

Version 0.27.3 adds four precision profiles, registered automatic and manual solver selection, floating-point reporting, unit-aware validation, condition-number governance, convergence warnings, uncertainty-standard metadata, and independent reference-method comparisons.

A new **Analyze → Precision & Solver Governance** workspace and `[sc_lab_solver_governance]` shortcode expose these controls through WordPress while the Python Compute Core remains the authoritative execution plane.
