# Install Sustainable Catalyst Lab v0.39.0

Keep these files together in `~/Downloads`:

- `sustainable-catalyst-lab-v0.39.0-upgrade-DIRECT-BRIDGE-V27.zip`
- `install_and_push_sustainable_catalyst_lab_v0_39_0_DIRECT_BRIDGE_V27_macos.sh`

Run:

```bash
cd ~/Downloads
chmod 700 install_and_push_sustainable_catalyst_lab_v0_39_0_DIRECT_BRIDGE_V27_macos.sh
./install_and_push_sustainable_catalyst_lab_v0_39_0_DIRECT_BRIDGE_V27_macos.sh
```

After the push, deploy the latest `main` commit in Render using **Clear build cache & deploy**. A persistent disk is recommended for institutional governance records.
