# Sustainable Catalyst Lab v0.35.1

## Review, Comments, Approvals, and Scientific Sign-Off

- Adds append-only, nested review comments with withdrawal records.
- Adds resource-scoped review threads and manager-controlled resolution or reopening.
- Adds reviewer assignments with revision-safe state transitions.
- Adds approval requests with configurable approval, open-thread, assignment, and signatory-role gates.
- Adds immutable approve, reject, request-changes, and abstain decisions with evidence and SHA-256 identities.
- Adds immutable scientific sign-off records binding the workspace, resource, approval, decisions, signatory, and statement.
- Migrates the shared workspace database from schema version 1 to schema version 2 in place.
- Adds authenticated FastAPI routes, logged-in WordPress proxies, eight public contracts, and the 69th Lab panel.
- Preserves all v0.35.0 workspace, membership, invitation, resource-link, ownership-transfer, archive, and access-decision behavior.
