# Scientific Source Registry

The release includes connectors for USGS FDSN earthquake events, NASA EONET v3 natural events, NASA Image and Video Library astronomy releases, OBIS API v3 marine occurrences, NCBI PubMed E-utilities, and the arXiv Atom API. NASA GIBS WMS supplies Earth-observation imagery. The interface records retrieval time, connector version, source terms notes, and source health. Users must review record-level licensing and scientific limitations before redistribution or consequential use.
