# Sustainable Catalyst Lab v0.12.0 — Civil Engineering and Infrastructure Systems

Adds 48 auditable methods across structures and construction materials, geotechnical engineering, hydrology and drainage, transportation engineering, water and wastewater systems, and infrastructure lifecycle, risk, reliability, resilience, and embodied carbon.

The release includes 18 deterministic browser benchmarks, Python parity, project and notebook records, visualization handoff, Report Composer compatibility, workspace backup/restore compatibility, and Decision Studio-ready analysis contracts.

Focused shortcode: `[sc_lab_civil_infrastructure]`

These methods are screening and educational calculations. They do not replace sealed design, site investigation, detailed structural/geotechnical/hydrologic/traffic/process models, code compliance, permitting, construction inspection, or licensed professional judgment.
