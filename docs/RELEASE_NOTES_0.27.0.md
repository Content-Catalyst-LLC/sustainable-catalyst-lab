# Sustainable Catalyst Lab v0.27.0

## Scientific Computing and Numerical Methods

- Added the Numerical Methods Studio under Analyze.
- Added 12 registered Python numerical methods covering root finding, adaptive integration, interpolation, ODEs, eigen analysis, optimization, FFT analysis, Monte Carlo uncertainty, bootstrap intervals, sensitivity, and parameter sweeps.
- Added automatic synchronous or queued execution through the existing Compute Core and persistent worker queue.
- Added governed example inputs, structured JSON editors, cancellation, project saving, accessible result plots, raw output, provenance, and JSON export.
- Added numerical catalog and health REST endpoints plus a focused shortcode.
- Added deterministic method tests and preserved all production recovery, lifecycle, accessibility, feed, and functional-validation protections from v0.26.6.
- Public arbitrary-code execution remains disabled.
