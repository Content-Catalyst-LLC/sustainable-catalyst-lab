# Python Compute Core Foundation — v0.26.0

Sustainable Catalyst Lab v0.26.0 promotes Python from an optional dispatcher to the platform's governed compute plane.

## Architecture

- WordPress/PHP remains the control plane and same-origin gateway.
- Browser JavaScript remains the interactive and lightweight computation layer.
- FastAPI becomes the registered scientific computation layer.
- Every canonical result carries a reproducibility and provenance manifest.

## Canonical routes

- `GET /health`
- `GET /v1/capabilities`
- `GET /v1/methods`
- `GET /v1/methods/{method_id}`
- `POST /v1/compute/run`
- `POST /v1/jobs`
- `GET /v1/jobs/{job_id}`
- `DELETE /v1/jobs/{job_id}`

The prior `/v1/execute`, `/v1/compare`, report, and Decision Studio validation routes remain compatible.

## Security

The backend accepts registered methods only. Requests can use timestamped HMAC-SHA256 signatures generated by WordPress. Legacy API-key authentication is retained for migration. Arbitrary public Python execution is not included.

## Included foundation methods

Mechanics, energy, biology, descriptive statistics, regression, linear algebra, polynomial roots, sampled integration, and a seeded Monte Carlo benchmark.

## Deployment boundary

This release provides a single CPU service and a process-local short-job queue. Durable queues, distributed workers, GPU workers, and persistent results are intentionally deferred to v0.26.1.
