# Sustainable Catalyst Lab v0.39.0 Release Notes

## Institutional Administration, Identity, and Governance

Sustainable Catalyst Lab v0.39.0 adds the institutional control plane for the connected scientific research platform.

### New capabilities

- Institution and organizational-unit registries
- Human and credential-free service principals
- Scoped institutional role bindings
- Workspace classifications and governance profiles
- Deterministic research-action policy evaluation
- Retention-policy references
- Quorum-based governance approvals
- Immutable approval decisions
- Hash-chained governance timelines and institutional dashboards
- WordPress Institutional Governance Console
- Python and TypeScript SDK governance methods

### Safety boundary

The release records identities and authority but does not issue credentials, store service-account secrets, perform SSO exchange, rotate encryption keys, or automate destructive data deletion.

### Compatibility

The cumulative DIRECT_BRIDGE_V27 installer accepts Sustainable Catalyst Lab repositories reporting v0.31.0 through v0.39.0 and preserves the v0.38.0 interoperability, v0.38.1 typed handoff, and v0.38.2 public integration contracts.
