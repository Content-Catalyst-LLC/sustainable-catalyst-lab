# Sustainable Catalyst Lab v0.35.0

## Shared Research Projects and Team Workspaces

This release adds role-governed collaboration around existing Lab resources. It includes private workspaces, stable actor identities, six membership roles, single-use invitations, ownership transfer, governed resource links, hashed access decisions, archive-only lifecycle management, and durable activity history. The WordPress panel is available to logged-in users, while the Python coordinator remains the source of truth for access decisions.
