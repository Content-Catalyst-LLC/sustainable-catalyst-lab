# v0.39.1 — Security, Privacy, Secrets, and Audit Hardening

Adds AES-256-GCM secret storage, versioned key rotation, scrypt service credentials, nonce replay protection, signed hash-chained audit events, security response headers, recursive privacy scanning/redaction, privacy-request workflows, SDK methods, and a WordPress Security & Privacy console. SSO and external KMS integration remain outside this release.
