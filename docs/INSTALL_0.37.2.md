# Install Sustainable Catalyst Lab v0.37.2

Place these files in `~/Downloads`:

- `sustainable-catalyst-lab-v0.37.2-upgrade-DIRECT-BRIDGE-V23.zip`
- `install_and_push_sustainable_catalyst_lab_v0_37_2_DIRECT_BRIDGE_V23_macos.sh`

Then run:

```bash
cd ~/Downloads
chmod 700 install_and_push_sustainable_catalyst_lab_v0_37_2_DIRECT_BRIDGE_V23_macos.sh
./install_and_push_sustainable_catalyst_lab_v0_37_2_DIRECT_BRIDGE_V23_macos.sh
```

The cumulative installer accepts repositories reporting v0.31.0 through v0.37.2.
