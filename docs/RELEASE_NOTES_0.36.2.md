# Sustainable Catalyst Lab v0.36.2 Release Notes

**Release:** Offline Field Research and Edge Synchronization

This release adds governed disconnected research packages, edge-device enrollment, signed resumable synchronization, cursor-based delta exchange, duplicate suppression, conflict records and resolutions, field-device provenance, and a dedicated operations panel.

The coordinator does not place restricted institutional data into field packages and does not execute arbitrary device code or unrestricted callbacks.
