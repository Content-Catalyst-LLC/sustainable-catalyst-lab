# Architecture and Building Performance

The v0.13.0 workspace connects architectural programming, envelope design, solar and daylight reasoning, indoor environmental quality, building energy, water, carbon, acoustics, and resilience in one traceable calculation environment.

Each result preserves:

- method and release identifiers
- equations
- labeled inputs and units
- labeled outputs and units
- assumptions and notes
- screening warnings
- validation-suite identifiers
- timestamps and audit fingerprints
- project and notebook handoffs
- visualization events

The workspace is intended for comparative reasoning, early design, education, transparent calculations, reproducible research, and documented scenario analysis.
