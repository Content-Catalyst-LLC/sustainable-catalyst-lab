# Install Sustainable Catalyst Lab v0.37.1

Place these files in `~/Downloads`:

- `sustainable-catalyst-lab-v0.37.1-upgrade-DIRECT-BRIDGE-V22.zip`
- `install_and_push_sustainable_catalyst_lab_v0_37_1_DIRECT_BRIDGE_V22_macos.sh`

Then run:

```bash
cd ~/Downloads
chmod 700 install_and_push_sustainable_catalyst_lab_v0_37_1_DIRECT_BRIDGE_V22_macos.sh
./install_and_push_sustainable_catalyst_lab_v0_37_1_DIRECT_BRIDGE_V22_macos.sh
```

The cumulative installer accepts repositories reporting v0.31.0 through v0.37.1.
